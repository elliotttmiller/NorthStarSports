Design Specifications

Basic Structure:

  • Border: 2px solid (various colors for states)
  • Border-radius: 8px
  • Padding: 12px 16px
  • Font-size: 1rem
  • Transition: all 0.2s ease

State Colors:

  • Default: Border #e0e0e0, Background white
  • Focus: Border #0066cc, Shadow rgba(0, 102, 204, 0.1)
  • Error: Border #dc3545, Background #fdf2f2
  • Success: Border #28a745, Background #f2f8f2
  • Disabled: Border #e0e0e0, Background #f1f3f4

Accessibility:

  • Minimum touch target: 44px height
  • Clear focus indicators with 3px shadow
  • Proper label association with for/id
  • ARIA attributes for error states
  • Color + text/icon for status indication
