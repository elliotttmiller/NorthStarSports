Usage Guidelines

Primary Buttons: Use for primary actions like "Place Bet", "Confirm", "Submit"

  • Only one primary button per screen section
  • Always use for the most important user action
  • Minimum touch target of 44px for mobile accessibility

Secondary Buttons: Use for secondary actions like "Cancel", "Back", "Skip"

  • Can have multiple secondary buttons per screen
  • Often paired with primary buttons
  • Less visual weight than primary buttons

Destructive Buttons: Use for irreversible actions like "Delete", "Remove", "Clear All"

  • Always confirm destructive actions
  • Use sparingly and with clear labeling
  • Consider adding confirmation dialogs
